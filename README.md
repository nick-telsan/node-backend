# Node Backend
Nodejs GraphQL API powered by Prisma and Apollo Server

## Getting Started
**Using Docker (recommended)**
`cp .env-example .env`
`docker-compose build`
`docker-compose up`

**Using Yarn**
Set up Postgres to your liking
`cp .env-example .env`
Update `DATABASE_URL` ind `.env` to match your Postgres database
`yarn`
`yarn dev`
